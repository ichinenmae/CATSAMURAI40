🎮 操作方法 (HOW TO PLAY)

シンプル操作で奥深い配送業務！



タップ（またはスペースキー）： ジャンプ

空中で もう一度押すと 2段ジャンプ！ 高い足場もこれで攻略だ。

長押し（またはスペースキー押し続け）： 急ブレーキ

画面スクロールを遅くして、敵や穴を慎重に回避せよ。

🚛 業務の流れ（MISSION）

【荷引き (GET CARGO)】

街に落ちている 「レジ袋」 を拾え！

最大3つまで積載可能。屋根に荷物が積まれていくぞ。

袋の色で報酬ランクが違う（白＜青＜金＜虹）。

【配達 (DELIVERY)】

荷物を持った状態で 「四角いドアの家」 を通過せよ！

通過するだけで配達完了。報酬ゲット！

まとめて届けると効率アップ！

【継続 (SURVIVE)】

警察やUFOに捕まらず、どこまで稼げるか挑戦だ！

走行距離に応じて「稼働時間」が記録されるぞ。

⚠️ 危険とペナルティ (DANGER)

🚓 パトカー & 🛸 UFO

接触すると即・業務終了（ゲームオーバー）。

【対抗策】 敵の真上から落下して 「踏みつけ (STOMP)」 れば無力化できるぞ！

🕳️ 落下事故

道路の切れ目に落ちると廃車確定。2段ジャンプで飛び越えろ。

🚪 誤配 (WRONG DOOR)

「上部が丸いドア」 は配達先じゃない！

間違えて届けると ペナルティ（報酬没収） だ。よく見て回避しろ！
